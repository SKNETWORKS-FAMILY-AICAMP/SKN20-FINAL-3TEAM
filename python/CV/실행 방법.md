## 실행 방법

### 환경 설정

```bash
# conda 환경 활성화
conda activate 5th_test

# 의존성 설치
pip install -r requirements.txt

# .env 파일 생성 (OpenAI API Key 설정)
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-4o-mini
OPENAI_TEMPERATURE=0.1
```

### 통합 실행 (권장)

```bash
# 사내 평가 문서 색인 (최초 1회만)
python run_inference.py --index-eval

# CV 추론 + RAG 분석 (전체 파이프라인)
python run_inference.py -i test_images/APT_FP_OBJ_001046197.PNG

# CV 추론만 실행
python run_inference.py -i test_images/APT_FP_OBJ_001046197.PNG --cv-only

# RAG 분석만 실행 (기존 topology.json 필요)
python run_inference.py -i test_images/APT_FP_OBJ_001046197.PNG --rag-only

# 배치 처리 (CV + RAG)
python run_inference.py -i test_images/ --batch
```

**출력:**
- `output/{이미지명}/topology_graph.json` (CV)
- `output/{이미지명}/analysis_result.json` (RAG)
- ChromaDB 색인

### 개별 실행 (선택)

```bash
# CV 추론만
python run_cv_inference.py -i test_images/APT_FP_OBJ_001046197.PNG

# RAG 분석만
python run_rag_inference.py --topology output/APT_FP_OBJ_001046197/topology_graph.json
```

### ChromaDB 내용 확인

```bash
python check_chromadb.py
```

**출력:** `chromadb_content.json` (디버깅용)
